Linear Applications & 2x2 Matrices

Chapter 1: Introduction to Linear Applications

Linear applications are functions between vector spaces that preserve vector addition and scalar multiplication. Here we will explore how to define and work with them.

In this chapter, we will discuss transformations, matrices, and how to represent them with matrices.

Chapter 2: Introduction to 2x2 Matrices

A 2x2 matrix is a grid of numbers arranged in two rows and two columns. Matrices are essential in linear algebra because they represent linear transformations. Here is an example of a 2x2 matrix:

 A = [ a  b ]
         [ c  d ]

Chapter 3: Matrix Multiplication

Matrix multiplication involves taking the dot product of rows and columns. Let's consider multiplying two 2x2 matrices:

 A = [ a  b ]    B = [ e  f ]
         [ c  d ]         [ g  h ]

Resulting Matrix C = AB is:

 C = [ ae + bg  af + bh ]
         [ ce + dg  cf + dh ]

Example: Matrix Multiplication

Let's try an example with specific values:

 A = [ 1  2 ]
         [ 3  4 ]   B = [ 5  6 ]
                       [ 7  8 ]

The result of A * B is:

 C = [ (1*5 + 2*7)  (1*6 + 2*8) ]
         [ (3*5 + 4*7)  (3*6 + 4*8) ]

Which gives:

 C = [ 19  22 ]
         [ 43  50 ]
